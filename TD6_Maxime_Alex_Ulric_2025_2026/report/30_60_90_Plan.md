# 30/60/90 Day Security Roadmap

This plan outlines the next steps to transition from the current hardened baseline to a mature security operations model.

| Timeframe | Strategic Actions |
|:---|:---|
| **30 Days (Quick Wins)** | • Implement automated key rotation policy for SSH.<br>• Schedule weekly security patching.<br>• Verify off-site backup integrity.<br>• Automate regression tests via cron jobs. |
| **60 Days (Architecture)** | • Deploy centralized log collection (ELK/Graylog).<br>• Begin SIEM integration for alert triage.<br>• Fine-tune WAF rules for web traffic.<br>• Perform deep segment reachability review. |
| **90 Days (Governance)** | • Migrate manual configs to Infrastructure as Code (IaC).<br>• Integrate security regression in CI/CD pipelines.<br>• Establish quarterly policy review cadence.<br>• Transition VPN from PSK to X.509 certificates. |
